Welcome to pubstack.

This developer environment is the easiest way to get started with all your publisher projects. All the tools require to get you up and running have already been set up for you.

Installation

  1. Install Vagrant.

  2. Install VirtualBox.

  3. Add project files to your machine (for example in ~Sites/www).

  4. Run the following:

    
git clone git@github.com:NBCUOTS/pubstack.git
cd pubstack
cp default.config.yml config.yml
# modify config.yml as needed (note each config has commented instructions):
vim config.yml
vagrant up
echo "172.20.20.10 pubstack" >> /etc/hosts

  5. Visit http://pubstack/ in your browser.
