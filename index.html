new Welcome to pubstack.

This developer environment is the easiest way to get started with all your publisher projects. All the tools require to get you up and running have already been set up for you.

Installation

  1. Install Vagrant.
  2. Install VirtualBox.
  3. Install Ansible:
    • Latest Releases Via Homebrew (Mac OSX)
    • Latest Releases Via Apt (Ubuntu)
    • See all
  4. Add project files to your machine (for example in ~Sites/www).

  5. Clone this repo:

    git clone git@github.com:NBCUOTS/pubstack.git
cd pubstack

  6. Create your config file from the default template and modify as needed (note each config has commented instructions):

    cp default.config.yml config.yml
vim config.yml

  7. Start-up your vagrant box:

    vagrant up

    NOTE: after the initial vagrant up, you will want to run vagrant provision any time you edit your config file values.

  8. Modify your /etc/hosts file with an entry for pubstack and any additional domains defined in the sites array in your config.yml file, for example:

    172.25.128.10 pubstack
172.25.128.10 local.site1.com
172.25.128.10 local.site2.com

  9. Visit http://pubstack/ in your browser.
