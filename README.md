pubstack
========

Publisher's DevStack

#Installation

1. Install [Vagrant](http://www.vagrantup.com).
1. Install [VirtualBox](https://www.virtualbox.org).
1. Run the following:

```bash
git clone git@github.com:NBCUOTS/pubstack.git
cd pubstack
cp Vagrantfile_user ~/.vagrant.d/Vagrantfile
vagrant up
```
